# Plugin Example 2

## Description
Adds new character customization options for RPG Maker MZ.

## Features
- Enhanced character customization.
- Fully compatible with RPG Maker MZ.

## Installation
1. Download the plugin file from [here](https://example.com/plugin2).
2. Place the file in your project's `plugins` folder.
3. Enable the plugin in the RPG Maker plugin manager.
